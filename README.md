# 垃圾厢房智能巡检演示

这是一个面向客户展示的网页演示，用公开固定点位截图呈现垃圾投放点巡检效果。

页面支持：

- 使用公开固定点位原始截图查看输入画面
- 使用同一公开系统的识别结果截图展示真实结果
- 模拟桶内高度、开盖记录、清运计划等现场联动信号
- 展示满溢、箱体定位、散落垃圾、复核建议和工单建议

公开演示地址：

https://densematrix-labs.github.io/waste-vision-demo/

样例来源：Semarang PantauSemar TPS BULU LOR 公开固定点位页面。
页面中的识别框来自公开结果截图，非前端人工绘制。
现场信号为演示联动而模拟，已在页面中明确标注。
